## 用户交互流程

### 1. 新建会话流程
```
点击 [新建会话] → 输入会话名称 → 选择AI工具 → 开始对话
```

### 2. 发送消息流程
```
输入消息 → 选择文件(可选) → 点击发送 → 等待AI响应 → 显示结果
```

### 3. 文件处理流程
```
拖拽文件或点击上传 → 文件预览 → 发送给AI → AI分析 → 显示分析结果
```

### 4. 设置配置流程
```
打开设置 → 修改配置 → 保存设置 → 重启应用 → 应用新配置
```

## 响应式设计

### 桌面端 (> 1024px)
- 完整的三栏布局
- 侧边栏显示完整会话列表
- 消息区域最大化

### 平板端 (768px - 1024px)
- 可折叠的侧边栏
- 消息区域自适应
- 优化的触摸交互

### 移动端 (< 768px)
- 单栏布局
- 底部导航栏
- 全屏消息界面
